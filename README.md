# Skyward Seeker

Skyward Seeker is a two-player platformer game built in Processing (Python mode). Compete to reach the gold, avoid obstacles, and collect power-ups to win!

## Features
- Two-player local gameplay
- Multiple levels with unique layouts and challenges
- Moving and static platforms
- Mines, bats, and falling obstacles
- Power-ups for extra lives and bullets
- Background music and sound effects
- Win by reaching the gold or outlasting your opponent

## Controls
**Player 1 (Left):**
- Move Left: A
- Move Right: D
- Jump: W
- Shoot: S

**Player 2 (Right):**
- Move Left: Left Arrow
- Move Right: Right Arrow
- Jump: Up Arrow
- Shoot: Down Arrow

## How to Run
1. Install [Processing 3](https://processing.org/download) and select Python mode (Jycessing).
2. Download or clone this repository.
3. Open `skywardseeker/finalyuheben/finalyuheben.pyde` in Processing.
4. Make sure the `images/` and `sounds/` folders are in the same directory as the sketch.
5. Click Run to start the game.
